# USIM Project

**USIM Project** es un proyecto base de Laravel preparado específicamente para ejecutar el ecosistema **USIM** en un entorno controlado y reproducible.

Este repositorio **no es un Laravel estándar**. Ha sido preconfigurado con una serie de decisiones técnicas que garantizan el correcto funcionamiento de USIM.

---

## Propósito

El objetivo de este proyecto es proporcionar un entorno inicial listo para:

- Ejecutar **USIM** sin configuración adicional compleja
- Estandarizar entornos de desarrollo
- Facilitar la reproducibilidad en contextos académicos y profesionales

---
